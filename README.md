# Nuwara-Eliya-Rainfall-Drivers
“Analysis of how air temperature, relative humidity, and wind speed influence rainfall in Nuwara Eliya, Sri Lanka

This dataset supports the abstract published at the Kandy Conference 2025 (https://www.kandyconference.org/), ID: ICSBE25_363, titled ‘Influence of Air Temperature, Relative Humidity, and Wind Speed on Rainfall Patterns in Nuwara Eliya, Sri Lanka’.”
